Selected Projects

Systems that combine LLMs, ML, and optimization.

LLM-Enhanced Forecasting with RAG-Style Retrieval

Built a forecasting pipeline where an LLM ingests match-day news, reports, and contextual data via retrieval and outputs structured signals (injury risk, morale, tactical changes). These features feed into a downstream Bayesian neural network that predicts outcome probabilities with calibrated uncertainty.

Impact: improved top-1 accuracy by ~8 percentage points over a feature-only baseline and reduced Brier score by ~20%, leading to better-calibrated probabilities for downstream decision rules.

    LLMs RAG-style retrieval PyTorch Bayesian NN
GitHub (forecasting pipeline)

Stochastic Routing Optimization for Bread Delivery

Two-stage stochastic TSP with time windows and random arc costs for a bakery delivery network. Plans a baseline route, then adapts to disruptions (traffic, closures) through recourse decisions, penalizing late or missed deliveries.

Impact: achieved roughly 13% reduction in expected delivery cost and cut late deliveries by ~25% compared to a deterministic heuristic baseline across sampled scenarios.

    Gurobi Stochastic Programming Routing Python
Project Report (PDF)

Facial Expression Detection (ResNet50–ViT Hybrid)

Hybrid ResNet50 + Vision Transformer model for facial expression recognition on low-resolution grayscale images, using transfer learning, augmentation, and cross-validation to improve robustness.

Impact: improved macro-averaged accuracy by ~6–8 percentage points over a plain CNN baseline and reduced overfitting on minority classes via better feature sharing.

    Computer Vision ResNet50 ViT PyTorch
GitHub (CV experiments)

News + Stock Forecasting & Trading Agent

Prototype that merges news-based embeddings with price history to forecast short-term price moves and drive a simple buy/sell agent. Uses a mixture-density-style output to capture multi-modal returns and backtests strategies against a naive baseline.

Impact: improved hit-rate on direction by ~7 percentage points over a price-only model and produced a higher Sharpe ratio in backtests under realistic transaction costs.

    NLP Time Series PyTorch Backtesting
GitHub (prototype)
