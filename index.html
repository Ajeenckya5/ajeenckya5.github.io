Selected Projects

Work that best represents how I think about AI/ML systems.

End-to-End Probabilistic Forecasting System

Designed a forecasting pipeline that combines LLM-derived event intelligence, match-day context, engineered features, and historical outcomes. Refactored a baseline neural network into a Bayesian architecture with Monte Carlo sampling to model predictive uncertainty and improve probability calibration.

    PyTorch Bayesian NN LLMs Calibration
GitHub (core codebase)

Stochastic Routing Optimization for Bread Delivery

Built a two-stage stochastic TSP with time windows and random arc weights for a bakery delivery network. The model plans a nominal route and then adapts to disruptions (traffic, closures) via scenario-based recourse decisions, penalizing late and missed deliveries.

    Gurobi Stochastic Programming Routing Python
Project Report (PDF)

Facial Expression Detection (ResNet50–ViT Hybrid)

Developed a hybrid ResNet50 + Vision Transformer pipeline for emotion classification on a large-scale facial expression dataset. Used transfer learning, extensive augmentation, and k-fold evaluation to robustly handle low-resolution grayscale images.

    Computer Vision ResNet50 Vision Transformer PyTorch
GitHub (CV experiments)

News + Stock Forecasting & Trading Agent

Experimental pipeline that merges news sentiment embeddings with price history to predict short-term price movements and drive a simple buy/sell agent. Focused on feature engineering, MDN-style outputs, and backtesting realistic strategies.

    NLP Time Series PyTorch Backtesting
