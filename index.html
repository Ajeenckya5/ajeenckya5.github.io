Built a probabilistic forecasting pipeline that unifies LLM-derived event intelligence, weather signals, engineered features, and historical data. Stochastic Routing Optimization for Bread Delivery

Two-stage stochastic TSP with time windows and random arc weights for a bakery delivery system. The model plans a baseline route and then adapts to real-world disruptions such as traffic or road closures using scenario-based rerouting and penalty structures for early/late deliveries.

Designed a hybrid ResNet50 + Vision Transformer pipeline on a large facial expression dataset (>300K images), using transfer learning and k-fold evaluation to robustly classify emotions from low-resolution grayscale inputs, outperforming CNN baselines.
